# Bacho Iliya - Next.js Application

🥛 Bulgarian Farm Products Interactive Website

## Project Structure
- **Root Directory**: This Next.js application is located in the ROOT directory
- **Framework**: Next.js 15 with App Router
- **Build**: `npm run build` 
- **Development**: `npm run dev`

## Deployment
- Built for Vercel deployment
- Production-ready Next.js application
- PostgreSQL database integration

## Important Notes
- ❌ NOT in `nextjs-app/` subdirectory
- ✅ Project files are in ROOT directory
- ✅ All configuration files (package.json, next.config.js) are in ROOT
